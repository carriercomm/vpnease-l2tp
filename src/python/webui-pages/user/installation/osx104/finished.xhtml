Using the VPN connection

You can open VPN connection from top right hand corner VPN menu:

  • Select Connect from the VPN menu.
  • If you have not save your password, a password dialog opens.
  • Enter your password and accept by clicking OK.
  • Ensure that connectivity to your company network resources works correctly by e.g. using your e-mail client or opening a web page from an intranet server.

You can disconnect the VPN connection by selecting Disconnect from the top right hand corner VPN menu.

Known limitations

  • When switching from one user to another, the VPN connection is automatically dropped by the operating system for security reasons.
  • When suspending and resuming (by closing laptop lid, for instance), the VPN connection will drop due to protocol timeouts. However, OS X will not notice this immediately when resuming. Disconnect and reconnect the VPN connection when resuming from suspend mode.
