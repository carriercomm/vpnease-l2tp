Server BIOS Setup

Overview

The PC BIOS controls many aspects of PC hardware relevant to use, such as computer startup procedure and low level hardware configuration. BIOS settings are specific to each motherboard, so the configuration procedure varies widely between BIOS versions.

server does not require any specific BIOS settings. The BIOS setup steps described below are recommended in order to minimize problems during the product installation and usage.

BIOS Setup Steps

We recommend the following BIOS setup steps.

BIOS Update

Check your BIOS version and install a BIOS update if available from your motherboard manufacturer. BIOS updates can prevent problems with operating system hardware drivers, and are especially important for recently introduced motherboard models.

BIOS Configuration

  • Enter BIOS configuration by booting your computer and pressing your BIOS specific settings activation key (typically DEL, F2, F8, or ESC)
  • Reset BIOS settings to default values
  • Configure processor, bus, memory, and performance settings. Use conservative settings (labeled Normal or similar) unless you have a good reason for doing otherwise
  • Configure boot order
    • Enable booting from the CD/DVD drive
    • Enable booting from USB HDD (for USB stick installation)
    • Enable booting from hard disk
    • Select boot order as follows: CD/DVD first, USB HDD second, and HDD last
  • Disable any automatic power management features
    • These include features such as screen savers, timed shutdown, etc
    • Such features are not desired in server systems as they may cause unpredictable operating system interactions and service disruptions
  • Disable extra devices
    • Disable any extra integrated audio, video capture, wireless LAN, or other unnecessary motherboard-specific devices
    • Integrated video (VGA) and integrated network interface cards (NICs) should be left enabled as appropriate
  • Disable anti-virus features (anti-virus detection and warning)
  • Set BIOS date and time
    • Note that will update its time automatically, but setting BIOS date and time is still recommended, as it minimizes system discrepancies (e.g. in log files)
    • Set BIOS time to UTC (Coordinated Universal Time), not your local time zone
  • Other settings
    • Enable Quick POST (Power On Self Test) if available; this speeds up the boot-up sequence
  • Save settings and exit BIOS setup

If any of your BIOS settings is out of the ordinary, write down any special settings for later use. This will help reconfigure the system should BIOS settings be lost due to motherboard NVRAM battery problems.

BIOS in Virtualization Products

Most virtualization products, such as VMware and Virtual PC, also have BIOS settings for the virtual machine hardware. Setting hardware and power management features is not crucial for such environments, as the defaults are usually acceptable for server use. However, boot order must still be configured correctly.
