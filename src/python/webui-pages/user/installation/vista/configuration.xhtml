Windows Vista prompts for administrative confirmation several times during the configuration process. You must confirm all actions in the configuration process. These confirmations are not described in the configuration process below.

Step 1: Modify Windows registry

  • In order to enable wider support for different kinds of networks, run the following registry file: vista-natt-enable.reg. For more information, see Microsoft KB article #926179. If you want to stop using later, you can undo the registry change by running the following registry file: vista-natt-disable.reg.
  • Reboot the computer after running the registry file for the change to take effect.

Step 2: Create VPN connection

  • Open Network and Sharing Center through Start → Control Panel → Network and Internet → Network and Sharing Center. (If you use Classic view, this path will be different.)
  • Click Set up a connection or network to start the connection creation wizard.
    • Select Connect to a Workplace and click Next.
    • If you have an existing connection of this type, Windows will prompt whether you want to use the existing connection or create new one. Select No, create a new connection and click Next.
    • Select Use my Internet connection (VPN).
    • If you have not defined an internet connection, Windows will prompt whether you want to set up an internet connection before continuing. If your computer is already connected to the internet, select I'll set up an internet connection later.
    • Enter the server address into the Internet address field (e.g. vpn.company.com or 123.123.123.123).
    • Enter a name for this profile into the Destination name field (e.g. My Company VPN).
    • Ensure that the Use a smart card checkbox is cleared.
    • Select whether you want to share this connection with other Windows Vista users. If so, select the Allow other people to use this connection checkbox, otherwise clear it.
    • Check the Don't connect now; just set it up so I can connect later checkbox.
    • Click Next.
    • Username and password dialog is shown. Click Create without entering any information ( username and password will be entered later).
    • Click Close to complete VPN connection setup.

Step 3: Configure VPN settings

  • Go to Network Connections through Start → Control Panel → Network and Internet → Network and Sharing Center → Manage network connections. (If you use Classic view, this path will be different.)
  • Select the newly created VPN connection (e.g. My Company VPN), right click to get the context menu, and select Properties.
  • Select Networking tab and do the following:
    • Change Type of VPN to L2TP IPsec VPN.
    • Click IPSec settings. Check Use pre-shared key for authentication and enter the pre-shared key of the server: . Click OK.
    • Select Internet Protocol Version 4 (TCP/IPv4) from the connection items list and click Properties. Click Advanced... and ensure that Use default gateway on remote network is selected.
    • Click OK to close the Advanced TCP/IP Settings dialog, then click OK to close the Internet Protocol Version 4 (TCP/IPv4) dialog.
  • Select the Options tab, click PPP Settings..., and check the Enable software compression checkbox. Click OK to close PPP Settings dialog.
  • Click OK to close VPN Connection Properties dialog.
  • configuration is completed.
