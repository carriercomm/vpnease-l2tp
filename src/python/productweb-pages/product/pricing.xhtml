Pricing

has a monthly per server license fee which depends on the number of concurrent end-user and site-to-site VPN connections as follows:

Number of concurrent end-user VPN connections
1-10 9€ / connection / month
11-100 8€ / connection / month
More than 100 7€ / connection / month
Number of site-to-site VPN connections
One site-to-site endpoint 19€ / connection / month

Note: for a site-to-site connection, two servers and thus two licenses are required (for a total of 38€ per site-to-site connection).

Pricing details

  • Monthly license fee is determined by the highest fee during a calender month. When updating server license, the updated monthly fee recurs at the start of every following month. If the license fee increases, the difference between current and new monthly fee is invoiced for the current month. If the license fee decreases, the difference between current and new monthly fee is not returned.
  • Each serves requires a separate license, and discounts (for end-user VPN connection count) apply to each server license individually.
  • Prices do not include VAT, see Payment Options for VAT details.
  • Pricing is subject to change with 3 months notice.
