This device contains an installed server. To activate the server, reboot the computer from this device. contains an integrated Linux distribution and does not need a host operating system.

Note that the device has been repartitioned by the installer. If you wish to erase from this device you need to repartition and reformat the device (this applies particularly to USB sticks). To do this, boot this Live CD and follow instructions to repartition the device.

Please do not modify data on this device unless you wish to erase the installation.
