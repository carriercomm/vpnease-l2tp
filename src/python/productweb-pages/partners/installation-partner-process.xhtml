Partner Process

How to Become a Partner?

To become a partner, you need to:

  • Accept the Partner License Agreement (PLA) and other legal obligations
  • Fill in and send a Request Partnership form
  • Receive a confirmation from the sales staff

Floating Licenses and License Activation

To make service installation and activation as convenient as possible for both installation and service provider partners, partners are supplied beforehand with "floating licenses". Floating licenses are directly functional but not yet associated with a specific customer. They allow installation and activation without a license request delay.

When you install a new server, use one of the (unused) floating license keys during installation. Then use the Activate License page to activate that license from floating to active mode. License keys must be activated within 7 days of installation. Unactivated floating licenses will be cancelled after 7 days of installation.

support will provide you with more floating license keys as you start to exhaust existing ones. If you need more floating license keys before that, please contact support.

Process for Installation Partners

The partner process for installation partners is very straightforward:

  • Install for a customer using a floating license key.
  • Activate the floating license using the Activate License page within 7 days of installation.
