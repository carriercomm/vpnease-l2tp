• Download and open the following configuration file to open a Network window with basic configuration settings for already filled in. If you wish to configure your VPN connection manually, click here.
  • Enter remote access information into the configuration fields:
    • Description: Name for the VPN connection (e.g. My Company VPN).
    • Server Address: server address (e.g. vpn.company.com or 123.123.123.123).
    • Account Name: Your username (e.g. john.doe).
    • User Authentication: If you want to save your password to this computer, enter your password here. If you leave the User Authentication field empty, your password will be asked every time the VPN connection is opened.
    • Machine Authentication: Your server pre-shared key is
  • Click OK to finish configuration.
  • The newly created profile is selected in the Configuration drop-down menu. Check the Show VPN status in menu bar checkbox to make the VPN connection accessible directly from the menu.
  • configuration is completed. Close the Internet Connect dialog.
