Features

Ease of Use

  • No VPN client software installation
  • Wide operating system support (e.g. Windows, Mac OS X, and Linux)
  • Automatic VPN configuration (Windows 2000, XP, and Vista)
  • Comprehensive network support (e.g. NAT, port forwarding, dynamic IP)
  • Flexible user authentication (RADIUS, Active Directory)
  • Automatic product updates (optional)
  • Installation on a dedicated server or a virtual machine (e.g. VMware, Virtual PC, Hyper-V, KVM)
  • Easy-to-use web management interface

VPN Connections

  • Remote user VPN connections (remote access VPN)
  • Connecting company sites (site-to-site VPN)

VPN Standard

  • L2TP/IPsec
  • Encryption: AES, 3DES, HMAC-MD5, HMAC-SHA1
  • Data compression
  • Machine authentication: pre-shared key
  • User authentication: username and password

User Authentication

  • Built-in database
  • External RADIUS servers with failover support
  • RADIUS enables support for a wide variety of authentication systems, including Active Directory (AD), SecurID and other similar tokens, LDAP, and UNIX user accounts
  • Built-in and external databases can be used simultaneously

Web-Based Management

  • SSL encrypted web interface
  • Separate web interfaces for administrator and VPN users
  • VPN user can change own password from the web interface (not available with RADIUS)
  • RADIUS/AD authentication also for web interface

Network and Application Support

uses the packet-based IPsec encryption standard, which means that all IP-based applications work through VPN connections without any software changes. In addition, has been designed to work with the widest variety of network configurations. In particular, supports:

  • NAT traversal: VPN connections work even when multiple clients reside behind the same NAT device
  • Port forwarding: server can be installed behind a firewall
  • Dynamic DNS: server can have a dynamic address, which is registered to a dynamic DNS provider
  • Data compression: compresses user data traffic, making the best use of even low bandwidth network connections
  • No packet fragmentation: avoids the use of fragmented IP packets to prevent compatibility problems with some networks
  • VPN traffic routing: direct, NAT, proxy ARP
  • VPN traffic firewall: connection oriented firewall for VPN traffic

Maintenance Features

  • SNMP monitoring: SNMPv1, SNMPv2c, VPNease MIB
  • Periodic restart
  • Automatic server software update (optional)
  • Monitoring of internal server state and automatic recovery
  • Monitoring of site-to-site VPN connections with automatic recovery

High Availability (HA)

  • High Availability (HA) installation option
  • Load balancing and redundancy with a cluster of multiple VPNease servers
  • DNS-based load balancing and recovery

Supported Server Platforms

  • 32-bit PC hardware (Linux compatible)
  • 256 MB RAM
  • 2 GB hard disk or USB Flash memory
  • One or more Ethernet network interfaces
  • Can also be installed to a virtual machine (VMware, Virtual PC, Hyper-V, KVM)
  • Installation from CD (ISO image)
  • A pre-installed VMware virtual appliance also available
