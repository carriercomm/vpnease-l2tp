SNMP Monitoring

Overview

supports read-only access to status data through SNMP (Simple Network Management Protocol) versions 1 and 2c. also has its own Management Information Base (MIB) for specific management information for use with an external monitoring program.

Management Information Base (MIB)

Download the MIB file from the download page.

Testing SNMP (Linux)

The following instructions are for testing SNMP access using the Net-SNMP tools.

Configure to allow SNMP access from the network interface(s) you choose and the SNMP community string (for authentication), from the License & Maintenance page. To show the specific SNMP values, issue the following command: 

$ snmptreewalk -v 2c -c public <server> 1.3.6.1.4.1.26058

To use more readable syntax and to get more readable output, you can download the VPNease MIB and configure Net-SNMP to use it.
