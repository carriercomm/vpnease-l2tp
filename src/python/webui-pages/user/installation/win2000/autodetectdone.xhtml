The autoconfiguration tool automatically creates a VPN connection profile and a desktop shortcut. It also ensures that your system configuration is compatible with . The following checks are made:

  • Existing VPN connection profiles for this server are replaced with the new profile.
  • A registry change is applied to enable remote access traffic encryption and create an IPsec encryption policy. For more information, see ProhibitIpsec.
  • Windows services required by VPN connections are enabled. The affected services are: IPSEC Policy Agent, Remote Access Connection Manager.
  • A reboot is required to activate the changes. After reboot, double click on the desktop icon to activate the VPN connection.

Please review Windows 2000 VPN limitations.

NOTE: If you have VPN client software installed, running the autoconfiguration tool may disable the VPN client and require a re-install of the VPN client. See VPN client compatibility information.
