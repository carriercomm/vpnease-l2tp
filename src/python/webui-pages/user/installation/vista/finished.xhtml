Using the VPN connection

  • Go to Network Connections through Start → Control Panel → Network and Internet → Network and Sharing Center → Manage network connections. (If you use Classic view, this path will be different.)
  • Start the VPN connection by double-clicking your connection profile (e.g. My Company VPN).
  • Enter your username and password to the dialog. If you want to save your username and password to this computer, check Save this user name and password for the following users checkbox and select either Me only or Anyone who uses this computer depending on your preferences. Click Connect.
  • Connection window will disappear and a tray icon will indicate that the connection is active.
  • When you connect for the first time, Windows opens a Set Network Location dialog which allows you to configure network location, which primarily affects connection security settings. Because the VPN connection is a trusted one, select the Work option. Click Close to complete network location configuration.
  • Ensure that connectivity to your company network resources works correctly by e.g. using your e-mail client or opening a web page from an intranet server.
  • You have several options to disconnect your VPN connection:
    • Click right mouse button over VPN connection icon and select Disconnect from the context menu.
    • Click taskbar connection icon. Select Connect or disconnect... from the appearing menu to open Connect to a network dialog. Select VPN connection (e.g. My Company VPN) and click Disconnect.

Optional: Creating a desktop shortcut

If you are using the VPN connection regularly you can create a shortcut for the VPN connection to your Windows desktop:

  • Go to Network Connections through Start → Control Panel → Network and Internet → Network and Sharing Center → Manage network connections.
  • Drag your VPN connection icon with right mouse button on the desktop.
  • Release mouse button and select Create Shortcuts Here from the appearing context menu.

Optional: Improving reliability

If you wish to improve the reliability of your VPN connection, you can enable automatic redialing. Click Properties from the connection dialog and select Options tab. Alter settings as follows:

  • Redial attempts → 50
  • Time between redial attempts → 10 seconds
  • Idle time before hanging up → never
  • Check the Redial if line is dropped checkbox

These settings ensure that Windows automatically reconnects if the server is temporarily unavailable. However, other settings may be more appropriate for your environment.
