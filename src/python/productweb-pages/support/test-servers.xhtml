Test Servers

The test server allows you to test client compatibility and to see the web interface for VPN users (there is a separate web interface for administrators) without installing server.

To use the web interface, click the test server URI below, and enter the test username and password below to login. To test the client compatibility, follow the client configuration instructions found from the web user interface. Use the test username, password and pre-shared key below to configure the client.

Parameter Value
Server web URI http://test.vpnease.com/
Server address test.vpnease.com
Pre-shared key psk
Test user name test
Test user password test

If you want to test server in your own environment, you can download the and test the server without registration. For more details, see Download.
