Step 1: Prerequisites

  • Ensure that you have administrator privileges. You need administrator privileges to install Windows updates and configure the VPN connection.
  • Ensure that you have the latest Windows 2000 Service Pack (SP4) installed. You can check your Windows Service Pack version from Start → Settings → Control Panel → System.
  • Ensure that you have downloaded latest Windows updates from Microsoft Update. Searching for Windows updates required by may not be possible without latest Windows updates.

Step 2: Install required Windows updates

requires High Encryption Pack (HEP) and Windows 2000 NAT traversal update to be installed.

  • Go to Microsoft Update site.
  • Select Use administrator options from Options menu.
  • Select search the Windows Update Catalog link found under Update multiple operating systems. The search page requires you to install an ActiveX component (Windows Update).
  • To find Windows 2000 NAT traversal update, enter 818043 to the search field and click Search. Add the Windows 2000 update to your basket.
  • To find High Encryption Pack (HEP), enter High Encryption Pack to the search field and and click Search. Add the Windows 2000 update to your basket.
  • Download and install the selected updates. Updates require computer reboot.

Step 3: Run autoconfiguration

To configure your VPN connection, select your operating system, click Autoconfigure, and run the downloaded executable (.EXE) file. If you wish to configure your VPN connection manually, click Configure manually.
