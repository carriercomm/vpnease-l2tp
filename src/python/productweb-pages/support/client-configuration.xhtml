Client Configuration

uses the built-in VPN client provided by your operating system, so no client installation is required. Configuring the built-in VPN client is easy: for Windows 2000, XP, and Vista, provides an automatic VPN connection configuration feature; for other operating systems provides easy step-by-step configuration instructions.

To configure you client for VPN use:

  • Start a web browser and type in the address of the server into the browser address bar.
  • Login using the username and password. Note that the same username and password are used for both web user interface and VPN access.
  • For Windows 2000, XP, and Vista, follow the instructions for automatic configuration.
  • For other operating systems, select the client operating system from the navigation menu on the left and follow configuration instructions.

You can preview the client configuration interface and optionally test compatibility with an actual client by using our test server. For more information about the test server, see Test Servers.

For some platforms, such as Linux-based systems, additional software is required. Such platforms do not typically have a built-in remote access VPN client, so additional software is mandatory.
