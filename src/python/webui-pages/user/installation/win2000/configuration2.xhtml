Step 3: Create VPN connection

  • Open Start → Settings → Control panel → Network and Dial-up Connections and double click Make new connection to open Network Connection Wizard.
  • You may be prompted for dialing information if you haven't configured it before. If you are not using a dialup connection, you can enter any information accepted by Windows. These settings have no effect on the VPN connection. You can enter the following information, for instance:
    • What country/region are you in now? → United States of America
    • What area code (or city code) are you in now? → 123
    • If you dial a number to access an outside line, what is it? → 123
    • The phone system at this location uses: → Tone dialing
  • Start the connection creation wizard by clicking Next.
  • Select Connect to a private network through the Internet and click Next.
  • You may be asked about initial connection dialing. If so, select Do not dial the initial connection and click Next. (If you are using GPRS or other dialup as a transport for your remote access connection, make the appropriate choice here).
  • Enter the server IP address into the Host name or IP address... field (e.g. 123.123.123.123). Note that you cannot use a DNS name as the server address in Windows 2000 due to Windows limitations. Click Next.
  • You may make the VPN connection available to only yourself or to all users. Select the appropriate choice and click Next.
  • Enter a name for the new VPN connection (e.g. My Company VPN), check Add a shortcut to this connection to my desktop, and click Finish to close connection creation wizard.
  • Windows opens the connection dialog automatically.

Step 4: Modify connection properties

  • Click Properties from connection dialog to open connection properties dialog.
  • Select Networking tab and change Type of VPN to Layer-2 Tunneling Protocol (L2TP).
  • Select Internet Protocol (TCP/IP) from the connection components list and click Properties to open Internet Protocol (TCP/IP) Properties dialog.
  • Click Advanced... to open Advanced TCP/IP Settings dialog and ensure that Use default gateway on remote network is selected.
  • Click OK to close the Advanced TCP/IP Settings dialog, then click OK to close the Internet Protocol (TCP/IP) Properties dialog.
  • Click OK to close the connection properties dialog, then click Cancel to close the connection window.
