A USB stick used as a installation target is not suitable for USB storage unless it is reformatted after use.

To proceed with formatting, select a target device to be formatted from the list below and click Next. The Live CD will repartition and reformat the device with the FAT filesystem, which is suitable for e.g. USB sticks.

The Live CD did not find a suitable formatting target. If you intend to format a USB stick, connect the stick now, wait for a few seconds for the device to be detected, and click Rescan.
