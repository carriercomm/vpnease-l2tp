Step 1: Activate VPN (first time only)

  • Open System Preferences and click Network.
  • Click + (plus sign) in the service list on the left. Select VPN from the Interface dropdown, ensure that VPN Type is set to L2TP over IPSec, and click Create. If there is already an L2TP VPN service on the list, you don't need to add a new one.

Step 2: Set-up VPN access configuration

  • Open System Preferences and click Network. Select the VPN service (VPN (L2TP)) from the service list on the left.
  • Select Add Configuration from the Configuration dropdown, enter a name for your VPN profile, and click Create.
  • Enter the following fields:
    • Server Address: server address (e.g. vpn.company.com or 123.123.123.123).
    • Account Name: Your username (e.g. john.doe).
    • Show VPN status in menu bar: Check this option.
  • Click Authentication Settings, and enter the following fields:
    • Password: If you want to save your password to this computer, enter your password here. If you leave the Password field empty, your password will be asked every time the VPN connection is opened.
    • Shared Secret: Your shared secret is
  • Click OK to close the Authentication Settings window.
  • Click Advanced, and check the Send all traffic over VPN connection checkbox. Without this setting you may experience various connectivity problems. Close the Advanced settings window.
  • Click Apply and close the Network window to finish configuration.
