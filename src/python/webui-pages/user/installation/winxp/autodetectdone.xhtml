The autoconfiguration tool automatically creates a VPN connection profile and a desktop shortcut. It also ensures that your system configuration is compatible with . The following checks are made:

  • Existing VPN connection profiles for this server are replaced with the new profile.
  • A registry change is applied to support a wider variety of networks. For more information, see Microsoft KB article #885407.
  • A registry change is applied to enable remote access traffic encryption. For more information, see ProhibitIpsec.
  • Windows services required by VPN connections are enabled. The affected services are: IPSEC Services, Remote Access Connection Manager.
  • Changes may require reboot on first configuration.

NOTE: If you have VPN client software installed, running the autoconfiguration tool may disable the VPN client and require a re-install of the VPN client. See VPN client compatibility information. Please ensure that you have the latest service pack installed.
