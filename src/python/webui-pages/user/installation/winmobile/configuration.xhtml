These instructions are for Microsoft® Windows Mobile™ Version 5.0 (OS 5.1.1702 Build 14366.1.0.1 on a HP iPAQ). There may be minor differences between operating system versions.

Step 1: Create VPN connection

  • Open Start → Settings → Connections → Connections.
  • Click Add a new VPN server connection. If you have already defined a VPN connection, click Edit my VPN servers and then click New.... We suggest you remove any unnecessary VPN connections before configuring a new VPN connection. You can delete a connection by tapping and holding over the connection name and selecting Delete from the menu that opens.
  • Enter a Name for the VPN connection, enter server address into the Host name/IP field, and set VPN type to IPSec/L2TP. Click Next.
  • Select authentication type A pre-shared key and enter the server pre-shared key: . Click Next.
  • Enter your username to the User name field and your password to the Password field. Leave the Domain field empty. Click Finish.

Step 2: Configure intranet addresses

To use the Windows Mobile VPN, you need to define which IP addresses and/or domain names (URIs) reside in your intranet. Windows Mobile VPN automatically opens the VPN connection when intranet resources are accessed. The Windows Mobile VPN differs from the desktop Windows VPN in this respect.

  • Open Start → Settings → Connections → Connections (should already be open).
  • Click Advanced → Exceptions....
  • For each intranet resource, do as follows: click Add new URL..., enter the IP subnet or domain name into the Work URL box and click ok. Finally, click ok to close the Exceptions dialog. You may use the following formats to specify resources:
    • All addresses: *.* (matches all IP addresses and domain names)
    • IP address: 196.168.0.1 (matches the 192.168.0.1 address)
    • IP subnet: 196.168.* (matches the 192.168.0.0/16 subnet)
    • Domain name: test.intra.example.com (matches the test.intra.example.com only)
    • Domain name wildcard: *.intra.example.com (matches all sub-domains of intra.example com, e.g. test.intra.example.com and long.test.intra.example.com)
  • Finally, click ok to close the connection settings dialog.
